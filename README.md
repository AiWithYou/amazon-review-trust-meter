# Amazon レビュー信頼度メーター

Amazon.co.jpの商品詳細ページで、商品情報・星別評価・画面に表示されているレビューを端末内で分析し、購入前に詳しく確認する必要度を表示するChrome拡張機能です。

Amazon、Amazon.co.jp、サクラチェッカーの公式・提携拡張機能ではありません。表示結果はレビューの真正性や不正を断定するものではなく、商品ページ上で追加確認したい兆候を示す参考情報です。

## v1.1の判定ロジック

独自注意度は0〜100です。点が高いほど注意が必要で、表示する信頼目安は `5 - 注意度 × 0.04` で★1.0〜5.0に換算します。

分析対象は次のとおりです。

- 商品情報：長すぎる商品名、商品名へのブランド名不記載、キーワード反復、強い宣伝表現、商品名と説明文の仕様値の矛盾
- 評価分布：★5への極端な集中、★5と★1への二極化、Amazon平均と星別分布の不一致
- 表示レビュー：120日以内への投稿日集中、高評価の短文・汎用表現、確認済み購入率、本文類似度

次の情報は取得していないため、判定に含みません。

- メーカー・ショップの過去履歴
- 削除されたレビューとレビュー件数の過去推移
- 公式サイトの有無やGoogle検索結果
- カテゴリ平均との差
- Amazonの商品ページに現在表示されていないレビュー

そのため、本家サクラチェッカーの点数ではなく、購入前の追加確認を促す独自指標です。

## 表示

商品名付近のAmazon評価・レビュー数の直下に、信頼目安、判定、独自注意度、「詳細を見る」だけをコンパクトに表示します。「詳細を見る」を開くと、取得項目、星別分布、判定根拠、点数の見方、本家サクラチェッカーへのリンクを確認できます。

購入ボックスの内部には挿入しないため、「1回限りの購入」と「定期おトク便」の間隔には影響しません。

## プライバシーと権限

- 商品ページの分析はブラウザ内で完結し、分析内容を外部APIや開発者のサーバーへ送信しません。
- `permissions`・`host_permissions`による追加API権限は要求せず、コンテンツスクリプトはAmazon.co.jpの商品詳細ページだけで動作します。
- 「本家サクラチェッカーで確認」を利用者がクリックした場合に限り、対象商品のASINを含むサクラチェッカーのページを新しいタブで開きます。
- アカウント情報、Cookie、注文履歴、支払い情報を読み取る処理はありません。

## 免責事項

- 独自注意度は、取得できた商品情報・星別分布・表示レビューをルールベースで評価した目安です。
- 誤検知や見逃しがあり得るため、購入判断は商品説明、販売元、低評価レビュー、返品条件なども確認したうえで行ってください。
- Amazon側のページ構造変更により、情報を取得できない、または表示位置が崩れる場合があります。
- 本ソフトウェアの利用によって生じた損害について、開発者は法令で認められる範囲で責任を負いません。

## ライセンス

現時点ではオープンソースライセンスを設定していません。リポジトリを公開しても、コードの再利用・改変・再配布を許諾するものではありません。再利用を認める場合は、目的に合うライセンスを別途追加してください。

## 欠損データの扱い

- 平均評価・レビュー数を含む4分類中3分類以上を取得できない場合、★を出さず「判定不能」と表示します。
- 取得できなかった項目を安全材料として減点しません。
- レビュー件数が多いことやAmazon発送であることだけでは、安全側へ加点しません。

## Chromeへの反映

1. `chrome://extensions` を開きます。
2. 「Amazon レビュー注意度メーター」の更新ボタンを押します。表示されない場合は「デベロッパー モード」をオンにします。
3. 対象のAmazon商品ページを再読み込みします。

初回導入の場合は「パッケージ化されていない拡張機能を読み込む」から、このフォルダを選択してください。
