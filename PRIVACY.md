# プライバシーポリシー

最終更新日: 2026年7月22日

「Amazon レビュー注意度メーター」（以下「本拡張機能」）は、Amazon.co.jpの商品詳細ページに表示されるレビュー関連情報と商品記載をユーザーの端末内で分析し、購入前の確認材料を表示するChrome拡張機能です。

## 1. 取り扱うデータ

本拡張機能は、ユーザーが開いた対応商品ページにおいて、機能提供に必要な次の情報をブラウザ内で読み取り、一時的に処理します。

- 対応するAmazon.co.jp商品詳細ページのURLとASIN
- 商品名、ブランド、商品説明、表示平均評価、レビュー件数、星別評価割合
- 現在ページに表示されている公開レビューの星評価、タイトル、本文、投稿日、購入確認、Vine表示、バリエーション、参考票、画像数
- 同一ページ内の重複投稿を確認するための、公開レビュープロフィールURLに含まれるレビュアー識別子

これらは、Chrome Web Storeのデータ分類では、ウェブサイトのコンテンツ、ウェブ閲覧アクティビティ、個人を特定できる情報の一部に該当する可能性があります。

## 2. 利用目的

上記データは、次の単一目的にのみ使用します。

- 商品ページ内で、レビュー兆候と商品記載の整合性をルールベースで分析する
- 補正評価、注意度、確認推奨などの判定と、その根拠を同じ商品ページ内に表示する

本拡張機能はAIや外部AI APIを使用しません。判定はブラウザ内の説明可能なルールベース処理で行います。

## 3. 保存、送信、共有

- 分析対象データを開発者のサーバー、外部AI API、解析サービス、広告サービスへ送信しません。
- 分析対象データをローカルストレージ、Chrome Storage、Cookie、IndexedDB、ファイルへ保存しません。
- 分析対象データは対応ページを表示している間だけメモリ上で処理され、ページの再読み込み、移動、またはタブを閉じることで破棄されます。
- 開発者は分析対象データを受け取らず、閲覧できません。
- データを第三者へ販売、貸与、広告利用、信用評価、融資判断に利用しません。

## 4. ユーザー操作による外部サイトへの移動

詳細欄の「本家サクラチェッカーで確認」をユーザーが明示的に押した場合に限り、対象商品のASINをURLに含む `https://sakura-checker.jp/` のページを新しいタブで開きます。この移動はユーザー操作によるもので、本拡張機能がバックグラウンドでデータを送信するものではありません。移動先サイトでのデータ取り扱いには、当該サイトのプライバシーポリシーが適用されます。

## 5. 権限と対象範囲

本拡張機能は、Manifestの `permissions` および `host_permissions` を要求しません。コンテンツスクリプトは、次のAmazon.co.jp商品詳細ページのURLパターンだけで実行されます。

- `https://www.amazon.co.jp/dp/*`
- `https://www.amazon.co.jp/*/dp/*`
- `https://www.amazon.co.jp/gp/product/*`

Amazonのトップページ、検索結果、注文履歴、アカウント、支払い情報、Cookie、ログイン情報にはアクセスしません。

## 6. リモートコード

本拡張機能はリモートコードを読み込み、実行しません。実行コードはすべて拡張機能パッケージ内に含まれます。

## 7. Chrome Web Store User Data Policyへの準拠

本拡張機能による情報の使用は、Chrome Web Store User Data Policy（Limited Use要件を含む）に準拠します。データは、ストア掲載情報とユーザーインターフェースで明示した単一目的の提供に必要な範囲でのみ使用します。

## 8. ユーザーによる管理

本拡張機能は設定データやユーザーデータを保存しません。Chromeの拡張機能管理画面から無効化または削除すると、以後の処理を停止できます。

## 9. ポリシーの変更

機能またはデータ取り扱いを変更した場合は、この文書を更新し、必要に応じてChrome Web Storeの掲載情報でも変更内容を開示します。
