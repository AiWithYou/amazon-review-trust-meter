# Chrome Web Store 掲載情報

最終更新日: 2026年7月22日  
対象バージョン: 2.1.0

Chrome Web Store開発者ダッシュボードへ入力する正本です。実装、README、プライバシーポリシーと矛盾しない内容を維持します。

## ストアの掲載情報

### 名前

```text
Amazon レビュー注意度メーター
```

名前は `manifest.json` から取得されるため、ダッシュボードでは原則変更しません。

### 概要

```text
Amazon.co.jpのレビュー兆候と商品記載を端末内で複合分析し、購入前の注意度と確認材料を商品ページに表示します。
```

### 詳しい説明

```text
Amazon.co.jpの商品詳細ページで、レビュー兆候と商品記載の整合性をブラウザ内で分析し、購入前に追加確認したいポイントを表示するChrome拡張機能です。

本拡張機能は、現在開いている対応商品ページのURL・ASIN、商品情報、星別評価、表示レビュー本文、投稿日、購入確認、Vine表示、参考票、公開レビュアー識別子などをユーザーの端末内で一時的に処理します。分析対象データを外部サーバーへ送信せず、保存もしません。

【主な機能】
・Amazonの評価欄直下に、補正評価、判定、注意度、詳細ボタンをコンパクトに表示
・星別評価の偏り、表示平均との整合性を確認
・レビュー本文の類似表現、短文・汎用表現の集中を確認
・14日、30日、90日単位の投稿集中を確認
・購入確認、Vine、参考票、同一ページ内の投稿者重複を複合分析
・商品名と商品説明にある容量、防水等級、連続時間などの記載矛盾を確認
・判定根拠を折りたたみ式の詳細欄に表示

【使い方】
1. 拡張機能をインストールします。
2. Amazon.co.jpの商品詳細ページを開きます。
3. Amazonの評価欄直下に表示されるカードを確認します。
4. 「詳細を見る」を押すと、分析材料と主な判定根拠を確認できます。

【判定について】
AIや外部AI APIによる判定ではありません。複数の弱い兆候が同じ方向に重なっているかを、ブラウザ内のルールベース処理で確認します。

注意度はレビュー不正や「サクラ」である確率ではありません。個々の投稿者や商品について不正を断定するものでもありません。表示レビューは全レビューの無作為標本ではないため、誤検知や見逃しがあり得ます。販売元、低評価レビュー、返品条件、商品説明なども併せて確認してください。

【プライバシー】
・分析はユーザーの端末内で完結します。
・分析対象データを開発者サーバー、外部AI、アクセス解析、広告サービスへ送信しません。
・分析対象データ、設定、閲覧履歴を保存しません。
・permissionsとhost_permissionsは要求しません。
・対応するAmazon.co.jp商品詳細ページだけで動作します。
・リモートコードを使用しません。

詳細欄の「本家サクラチェッカーで確認」をユーザーが押した場合だけ、対象商品のASINを含む外部サイトを新しいタブで開きます。

【対応ページ】
Amazon.co.jpの商品詳細ページに対応しています。トップページ、検索結果、注文履歴、他国のAmazonサイトでは動作しません。

本拡張機能は、Amazon、Amazon.co.jp、Amazon Vine、サクラチェッカーの公式または提携拡張機能ではありません。
```

## プライバシーへの取り組み

### 単一目的

```text
Amazon.co.jpの商品詳細ページに表示されるレビュー関連情報と商品記載をユーザーの端末内で分析し、購入前の確認材料として補正評価、注意度、判定根拠を同じ商品ページ内に表示することです。
```

### サイトアクセスの理由

ダッシュボードにAmazon.co.jpへのアクセス理由が表示された場合は、次を入力します。

```text
対応するAmazon.co.jp商品詳細ページから、商品名、商品説明、表示平均評価、レビュー件数、星別割合、現在表示されている公開レビュー情報を読み取り、ユーザーの端末内で注意度と確認材料を計算して同じページに表示するために必要です。対象URLはmanifestのcontent_scripts.matchesに記載した商品詳細ページだけに限定しています。データの外部送信や保存は行いません。
```

### リモートコード

```text
いいえ、リモートコードを使用していません。
```

補足欄がある場合:

```text
実行するJavaScriptとCSSはすべて拡張機能パッケージ内に含まれています。外部スクリプト、WebAssembly、eval、外部AI API、CDN上の実行コードを読み込みません。
```

### 取り扱うユーザーデータ

Chrome Web Storeの定義ではローカル処理も申告対象になるため、次を選択します。

- 個人を特定できる情報: 同一ページ内の重複確認に使う公開レビュアー識別子
- ウェブ履歴またはウェブ閲覧アクティビティ: 現在開いている対応商品ページのURLとASIN
- ウェブサイトのコンテンツ: 商品情報、星別評価、現在表示されている公開レビュー

次は選択しません。

- 健康情報
- 財務、支払い情報
- 認証情報
- 個人的なコミュニケーション
- 位置情報
- ユーザー操作の追跡

### データ利用に関する宣誓

表示されるLimited Use関連の宣誓にはすべて同意できます。

- データを本拡張機能の明示した単一目的以外に使用または移転しない
- データを広告、信用評価、融資判断に使用または移転しない
- データを第三者、データブローカー、情報再販業者へ販売しない
- 開発者や第三者がデータを人手で閲覧しない

### 審査担当者向け手順

```text
1. 拡張機能をChromeへインストールします。
2. Amazon.co.jpの任意の商品詳細ページ（URLに /dp/ または /gp/product/ を含むページ）を開きます。
3. すでに商品ページを開いていた場合はページを再読み込みします。
4. 商品名付近にあるAmazonの評価欄の直下に「補正評価・判定・注意度・詳細を見る」が表示されることを確認します。
5. 「詳細を見る」を押し、分析材料、主な判定根拠、判定方法、免責事項が展開されることを確認します。
6. 詳細を閉じられること、ページ操作を妨げないことを確認します。

本拡張機能はAmazon.co.jpの商品詳細ページ以外では動作しません。ユーザーデータの外部送信、保存、リモートコード、ログイン、課金、広告はありません。
```

## 配布

### アイテムの公開設定

初回公開の方針を確認したうえで、次のいずれかを選びます。

- 一般公開する場合: 公開
- URLを知っている人だけに配布する場合: 限定公開
- 指定テスターだけの場合: 非公開

### 地域

本拡張機能はAmazon.co.jp専用かつ日本語UIのため、初回は日本を対象地域とするのが最も正確です。将来ローカライズする場合に対象地域を追加します。
